# **Deep Learning**
---
> Deep Learning experiments that use the two main frameworks in the field, `tensorflow` and `torch` as well as their derivatives like `keras` and `torchvision`.

----------
Each subfolder is named after a framework, and within them, there are further subfolders for the main tasks in the field using that particular framework.

## Technical details
- Medium to high memory and storage space. 
- Most of them requires **GPU access**.
